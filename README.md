# Digital Pet Care System 🐾

## 📌 Overview
The Digital Pet Care System is a C++ application designed to manage pet information efficiently. It helps users store and track pet details such as name, type, age, vaccination, and feeding schedule.

## 🚀 Features
- Add new pet details
- View all pets
- Search for a pet
- Delete pet records
- File handling for permanent storage

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File Handling

## 📂 Project Structure
- pet.h / pet.cpp → Pet class and functions
- file.h / file.cpp → File operations
- main.cpp → Menu-driven program

```bash
g++ main.cpp pet.cpp file.cpp -o PetCareApp
